# IC Design Project Portfolio

This repository contains a collection of independent integrated circuit
and digital system design projects completed during my undergraduate studies
in Electronic Information Engineering.

All projects were developed individually and focus on digital IC design,
memory circuits, compute-in-memory architectures, and FPGA-based system
integration.

## Project Overview

### Project 01 – 6T SRAM Compute-in-Memory Array
- 16×16 SRAM array with adaptive write-assist and write-skip control
- Transistor-level simulation (HSPICE)
- SNM analysis and CIM functionality

### Project 02 – Dual-UART Asynchronous Bridge
- Multi-baud UART bridge with FIFO buffering
- Dynamic channel switching and CDC handling
- FPGA verification

### Project 03 – Reusable Digital IP Library
- UART, SPI, I2C, AXI-Lite, HDMI timing, FIFO
- Parameterized Verilog modules with testbenches

### Project 04 – Full-Custom 8-bit Tree Adder
- Transistor-level gate design
- Full layout with DRC/LVS/PEX
- Post-layout timing and power analysis

Each project folder contains detailed documentation, design files, and
simulation results.
